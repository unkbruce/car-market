1. 프로젝트 개요

1.1 프로젝트명

실시간 Car Market

1.2 개발 목적

구매자와 딜러가 온라인에서 중고차 매물을 확인하고, 차량 상세 정보와 이미지를 바탕으로 실시간 상담까지 이어갈 수 있는 중고차 거래 지원 서비스를 구현한다.

1.3 서비스 소개

실시간 Car Market은 구매자가 차량을 검색, 필터링, 정렬하여 원하는 매물을 찾고, 딜러와 실시간 상담을 진행할 수 있는 웹 서비스이다. 딜러는 본인 차량을 등록, 수정, 삭제할 수 있으며 직접 업로드 이미지와 프로젝트 내 샘플 이미지를 함께 활용할 수 있다.

1.4 개발 배경

중고차 탐색 과정에서는 차량 조건 비교, 이미지 확인, 판매자 상담이 연속적으로 필요하다. 본 프로젝트는 차량 조회부터 실시간 상담까지 하나의 흐름으로 연결하고, 딜러의 매물 관리와 구매자의 상담 경험을 단순화하는 것을 목표로 한다.

1.5 핵심 사용자 시나리오

사용자시나리오
구매자회원가입 또는 로그인 후 차량 목록에서 조건을 선택하고, 상세페이지에서 차량 정보를 확인한 뒤 상담하기를 통해 딜러와 대화한다.
딜러딜러 권한으로 로그인 후 차량을 등록하고, 본인 차량을 수정/삭제하며, 구매자가 만든 상담방에서 실시간 메시지를 주고받는다.

2. 사용자 유형

사용자 유형설명이용 가능 기능
구매자차량을 탐색하고 딜러에게 상담을 요청하는 사용자회원가입, 로그인, 차량 목록/상세 조회, 검색/필터/정렬, 상담방 생성, 실시간 상담, 상담방 나가기

NFR-05검색 및 필터는 조건 변경 후 즉시 반영되어야 한다.구현 완료

NFR-07React Router를 사용하는 정적 사이트 배포에서 /* 요청은 /index.html로 rewrite되어야 한다.구현 완료
NFR-08Render 무료 서버의 로컬 파일 시스템은 재배포, 재시작, 인스턴스 교체 시 업로드 이미지가 사라질 수 있음을 안내해야 한다.문서 반영
NFR-09환경 변수 실제 값과 비밀정보는 문서와 저장소에 노출하지 않아야 한다.문서 반영
